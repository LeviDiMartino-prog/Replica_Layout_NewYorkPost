# Replica_Layout_NewYorkPost
Le numerose animazioni create con JavaScript nel sito originale (https://nypost.com/), come da consegna non sono state replicate.
Ho preferito condensare tutti gli stili in un singolo file .css, usando dei commenti per separare le varie sezioni e orientarsi.
Non ho sostituito con "Lorem ipsium" il testo scritto che, a intuito, non dovrebbe essere variabile nel tempo.
